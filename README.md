Simple classic Minesweeper game made with python and pygame gui module.

I. REQUIREMENTS:
+ PYTHON 3.8+
+ MODULES: **pygame** AND **numpy**
II. HOW TO RUN:
+ DOWNLOAD REPO, UNZIP AND GO TO PROJECT'S FOLDER THROUG CONSOLE
+ RUN GAME WITH:
```
python ./run.py
```
+ ENJOY!
